# ReflowOven

Cílem projektu je postavit přetavovací pec na plošné spoje.

Projekt byl současně zpracováván jako školní projekt do předmětu Vestavné systémy v roce 2026.

Projekt je postaven na platformě ESP32. Program řídí operační systém FreeRTOS.

Pec je postavena z obyčejné kuchyňské trouby. Pro ohřev byl naimplementován PID regulátor, který řídí spouštění topných tyčí.
Pro zařízení byla navržena i jednoduchá aplikace pro vzdálené ovládání.

Prototyp zařízení už je hotový, projekt ale ještě není ukončen a bude se na něm dále pracovat.
